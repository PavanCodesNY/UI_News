## 1. Wolverine Worldwide — https://www.awwwards.com/sites/wolverine-worldwide

WHAT IT DOES: Full-bleed GSAP scrollytelling spine that cross-fades between four brand worlds (Merrell, Saucony, Sperry, Hush Puppies). The layout pins a single canvas while a horizontal track translates X via ScrollTrigger; within each "world," product hero images parallax against gradient washes, and brand wordmarks scale-morph using SVG `filter: url(#displacement)` turbulence. Visual hierarchy: oversized serif display type sits over silent video b-roll, with hairline 12px monospace metadata captions anchored bottom-left as a persistent overlay.

PACKAGES:
- `next@15`
- `gsap@3.13`, `@gsap/react`, `gsap/ScrollTrigger`, `gsap/Observer`, `gsap/SplitText`
- `lenis@1.1` (smooth scroll)
- `framer-motion@11` (UI micro-interactions only)
- `tailwindcss@4`
- `@vercel/og` (dynamic OG)
- Fonts: `next/font/local` with Editorial New (serif display) + JetBrains Mono (caption)

IMPLEMENTATION:
1. Mount `<Lenis options={{ lerp: 0.08, wheelMultiplier: 1.0 }} />` at root, then `gsap.ticker.add((t)=>lenis.raf(t*1000))`.
2. Build a single `<section data-pin>` that is `100svh × 400svh` tall. Inside, an absolute container of width `400vw` translates X with `gsap.to(track, { xPercent: -75, ease: 'none', scrollTrigger: { trigger: section, pin: true, scrub: 1.2, end: '+=300%' } })`.
3. Each "world" panel = `100vw × 100svh` flex column with: hero product image (60% width, right-aligned, `mix-blend-mode: lighten`), oversized brand mark (font-size `clamp(8rem, 18vw, 22rem)`, `letter-spacing: -0.04em`).
4. Wordmark morph: wrap each letter via SplitText `splitText.chars`, drive `gsap.fromTo(chars, { yPercent: 110, rotate: 8 }, { yPercent: 0, rotate: 0, stagger: 0.025, ease: 'expo.out', duration: 1.2 })`.
5. Persistent metadata bar: fixed `bottom-6 left-6`, format `WORLD 02 / 04 — SAUCONY — 1898`, animate text swap on world change with `gsap.to(el, { opacity: 0, y: -8, duration: 0.3 })` → swap text → reverse.
6. Background gradient per world (CSS variable `--world-bg`): `gsap.to(':root', { '--world-bg': worldColor, ease: 'power2.inOut' })`. Each world declares `data-bg="#0B0F1A"`.
7. Responsive: below 1024px, kill horizontal pin and stack vertically with `ScrollTrigger.matchMedia`.
8. Perf: `will-change: transform` on track, preload images via `<link rel="preload" as="image">`, decode async, use AVIF.

WHY IT WORKS: Hick's law applied to navigation — by collapsing four brands into a single linear track, decision load disappears. The oversized type acts as a visual anchor (Gestalt figure/ground) while monospace metadata signals craft. The horizontal-on-vertical-scroll mapping creates kinesthetic novelty without losing intuitive direction.

GOOD / BAD:
- GOOD: Single canvas + four worlds = stunning brand cohesion at zero cognitive cost.
- BAD: Pinned horizontal scroll on a hard `scrub: 1.2` makes "skim reading" impossible; users hunting a specific brand must traverse all four.

CLEAN APPLICATION: Apply this to Clean's landing-page hero by replacing the four brand worlds with four agent-runtime contexts ("Cursor," "Claude Code," "Aider," "Custom"). Same horizontal-pin architecture. Each panel shows the same Clean proxy diagram but with that runtime's tool list streaming through — visualizing the universal context layer concept with one continuous motion instead of four screenshots.

## 2. Sui — https://www.awwwards.com/sites/sui

WHAT IT DOES: Three.js object-centric hero where a translucent geometric "object" (extruded hexagon with refraction shader) rotates on mouse-parallax while particle field flows beneath. Below the fold, a horizontal feature grid uses `position: sticky` + `IntersectionObserver` to swap the 3D scene's MeshTransmissionMaterial parameters as the user scrolls through "parallel execution," "sub-second finality," "object model." Monospace numerics animate via tween counters; dark navy ground (#0A0E1A) with cyan-violet gradients (#6FBCFF → #B58CFF).

PACKAGES:
- `next@15`, `react@19`
- `three@0.168`, `@react-three/fiber@9`, `@react-three/drei@10`, `@react-three/postprocessing@3`
- `three-mesh-bvh` (raycast perf)
- `framer-motion@11`
- `gsap@3.13` + `ScrollTrigger`
- `lenis@1.1`
- `tailwindcss@4`
- Fonts: `next/font/google` Inter Tight + IBM Plex Mono

IMPLEMENTATION:
1. Canvas: `<Canvas dpr={[1, 1.75]} gl={{ antialias: true, alpha: true }} camera={{ position: [0, 0, 4], fov: 35 }}>`. Use `<Preload all />` and `<AdaptiveDpr pixelated />`.
2. Hero object: extruded hex via `<mesh>` with `<MeshTransmissionMaterial samples={6} thickness={0.8} chromaticAberration={0.05} anisotropy={0.3} roughness={0.05} ior={1.4} backside />`. Lights: 1 `directionalLight` (intensity 1.2, position [4,4,4]) + `<Environment preset="city" />` for IBL.
3. Mouse parallax: `useFrame((s)=>{ mesh.rotation.y = THREE.MathUtils.lerp(mesh.rotation.y, s.pointer.x*0.4, 0.06); mesh.rotation.x = THREE.MathUtils.lerp(mesh.rotation.x, -s.pointer.y*0.3, 0.06) })`.
4. Particle field: `<Points>` of 8000 vertices in a ring distribution, custom shader material that animates `aOpacity` based on distance from camera. Use `THREE.AdditiveBlending`.
5. Scroll-driven material swap: `<ScrollControls pages={4}>` from drei; in `useScroll()` callback, interpolate `thickness`, `chromaticAberration`, and bg gradient stops.
6. Number tickers: framer-motion `useMotionValue` with `animate(mv, target, { duration: 1.4, ease: [0.16,1,0.3,1] })`, render via `mv.on('change', v => el.textContent = Math.round(v).toLocaleString())`.
7. Mobile fallback: detect `window.matchMedia('(max-width: 768px)')` → swap canvas for static AVIF render + scroll-pinned CSS gradient for the same effect.
8. Postprocessing: `<EffectComposer><Bloom intensity={0.6} luminanceThreshold={0.6} mipmapBlur /><Noise opacity={0.04} /></EffectComposer>`.

WHY IT WORKS: The refractive object IS the brand metaphor (Sui = objects, not accounts) — semantic motion. Refraction enforces depth perception (atmospheric perspective) which encodes "scalable infrastructure" subconsciously. Cyan-violet on near-black satisfies the WCAG AA contrast budget while reading as "premium tech" (BMW, Stripe, Linear all share this register).

GOOD / BAD:
- GOOD: 3D primitive doubles as logo — every frame reinforces positioning.
- BAD: MeshTransmissionMaterial at `samples={6}` melts mid-tier GPUs; mobile fallback feels static by comparison.

CLEAN APPLICATION: Apply this to Clean's product-demo section by rendering the MCP proxy as a single transmissive 3D object — tools "pass through" the object and emerge as TOON-formatted output, visualized as refracted light. Use the same dark navy + cyan accent. The object becomes a literal proxy metaphor.

## 3. EverSwap — https://www.awwwards.com/sites/everswap

WHAT IT DOES: DeFi protocol marketing site with a Three.js floating-token cluster (procedural mesh + emissive shader) that orbits a central "liquidity ring." Two-column above-the-fold: left = oversized headline with masked gradient text fill (`background-clip: text`), right = the 3D scene. Below: a fixed left-column nav tracks scroll progress through a vertical SVG path that "fills" as you scroll, with section dots at named anchors. Color: deep blue-black (#080A14), accent neon green (#9DFFB0), text greys.

PACKAGES:
- `next@15`, `three@0.168`, `@react-three/fiber@9`, `@react-three/drei@10`
- `gsap@3.13`, `ScrollTrigger`, `MotionPathPlugin`
- `lenis@1.1`
- `motion@11` (framer-motion successor)
- `clsx`, `tailwind-merge`
- `tailwindcss@4`
- Fonts: Aeonik Pro (display), JetBrains Mono (numerics)

IMPLEMENTATION:
1. Hero scene: cluster of 7 floating tokens (low-poly icosahedrons, 0.5 unit radius). Each has `<meshStandardMaterial emissive={tokenColor} emissiveIntensity={1.2} metalness={0.6} roughness={0.2} />`. Orbital motion: `useFrame((s,d)=>{ tokens.forEach((t,i)=>{ t.position.x = Math.cos(s.clock.elapsedTime*0.3 + i)*radius; t.position.z = Math.sin(s.clock.elapsedTime*0.3 + i)*radius; t.rotation.y += d*0.4 }) })`.
2. Central ring: `<torusGeometry args={[1.6, 0.02, 16, 96]} />` with `meshBasicMaterial color="#9DFFB0"` + emissive bloom.
3. Headline mask-fill: `<h1 className="text-[clamp(4rem,9vw,11rem)] leading-[0.85] tracking-[-0.04em]">` with `style={{ backgroundImage: 'linear-gradient(180deg,#fff 0%,#9DFFB0 100%)', WebkitBackgroundClip: 'text', color: 'transparent' }}`.
4. Sticky progress nav: `position: fixed; left: 24px; top: 50%; transform: translateY(-50%)`. SVG `<path>` with `stroke-dasharray` and `stroke-dashoffset` driven by `ScrollTrigger.create({ start: 'top top', end: 'bottom bottom', onUpdate: (s) => path.style.strokeDashoffset = pathLength * (1 - s.progress) })`.
5. Section dots: `IntersectionObserver` on each anchor; active dot gets `scale(1.4)` and `box-shadow: 0 0 24px #9DFFB0`.
6. Postprocessing: `<EffectComposer><Bloom intensity={1.1} luminanceThreshold={0.4} mipmapBlur /><ChromaticAberration offset={[0.0008,0.0008]} /></EffectComposer>`.
7. Scroll-locked cinematic: at section 2, camera `useFrame` lerps `position.z` from 5 → 2.5, creating a dolly-in as numbers tick.
8. Perf: cap DPR `[1, 1.5]`, freeze emissive on mobile, prefer static screenshot if `navigator.connection.effectiveType === '3g'`.

WHY IT WORKS: Neon green on black is signal-rich (terminal heritage, finance ticker) → reads "we are infrastructure." The orbital cluster encodes the unified-liquidity narrative more economically than any diagram. The sticky path nav addresses the long-scroll wayfinding problem without adding chrome.

GOOD / BAD:
- GOOD: The sticky SVG progress path doubles as nav AND as a brand element — zero wasted pixels.
- BAD: Emissive bloom + chromatic aberration at this intensity strains accessibility — contrast on the green text vs background fluctuates under bloom.

CLEAN APPLICATION: Apply this to Clean's pricing/protocol page by replacing the floating-token cluster with floating "tool definitions" (function signatures rendered as low-poly cards) orbiting a central Clean proxy ring. The sticky SVG progress path is a verbatim transplant — use it on Clean's docs to show user progress through a "Getting Started" chapter.

## 4. Indigo Laboratory — https://www.awwwards.com/sites/indigo-laboratory

WHAT IT DOES: Fashion/music project from FPTP Studio + Marea using Curtains.js to convert DOM `<img>` and `<video>` elements into WebGL textured planes, then warp them via fragment shaders on hover. The layout is a vertical-scroll editorial grid; on cursor-enter each image triggers a ripple displacement shader (perlin noise + sin wave), while text labels split into chars and float upward with stagger. Built on Nuxt 3 + GSAP. Color palette: bone white background (#F4F1EC), indigo (#1B1F4A), warm red accent (#E0563F).

PACKAGES:
- `nuxt@3.13`, `vue@3`
- `curtainsjs@8.1.5`
- `gsap@3.13` + `ScrollTrigger` + `SplitText`
- `lenis-vue@1.1`
- `@vueuse/core@11`
- Fonts: ABC Diatype (sans), Romana (serif)

IMPLEMENTATION:
1. Initialize Curtains: `const curtains = new Curtains({ container: 'canvas-wrapper', pixelRatio: Math.min(1.5, devicePixelRatio), watchScroll: true, autoRender: false })`. Use a single `Renderer` per page for perf.
2. Wrap each media element: `<div class="plane" data-vs="vs/ripple.glsl" data-fs="fs/ripple.glsl"><img src="..."></div>`. Create a `Plane` per `.plane` with `widthSegments: 20, heightSegments: 20, uniforms: { uTime: { value: 0 }, uHover: { value: 0 }, uMouse: { value: [0,0] } }`.
3. Fragment shader: classic perlin displacement `vec2 uv = vUv; uv.x += sin(uv.y*10. + uTime)*0.02*uHover; gl_FragColor = texture2D(uTexture, uv);` — multiply displacement by `uHover` (0→1 lerp on mouseenter/leave).
4. Mouse-enter event: `gsap.to(plane.uniforms.uHover, { value: 1, duration: 0.8, ease: 'expo.out' })`. On leave reverse.
5. RAF: `function tick(){ curtains.render(); requestAnimationFrame(tick) }`; integrate with Lenis so scroll position drives `plane.updatePosition()`.
6. Editorial typography: serif headline `font-size: clamp(3.5rem, 8vw, 9rem); line-height: 0.9; letter-spacing: -0.02em`. Mix serif + sans within the same line for editorial rhythm.
7. SplitText on labels: `const split = new SplitText(label, { type: 'chars,words' }); gsap.from(split.chars, { yPercent: 110, stagger: 0.012, ease: 'expo.out', duration: 0.9, scrollTrigger: { trigger: label, start: 'top 85%' } })`.
8. Cursor: custom 12px circle that morphs to 80px with mix-blend-mode `difference` over media planes.

WHY IT WORKS: Editorial typography + shader-warped imagery sits in the negative space of "developer tools" — it tells the user "we are not Material UI." The ripple shader on hover gives weight to imagery as if it were physical, which is the affordance prestige brands rely on.

GOOD / BAD:
- GOOD: Curtains.js is a 70KB DOM-aware WebGL bridge — half the bundle of a R3F equivalent.
- BAD: Custom cursor + shader hover + scroll-driven type is unusable on touch; the mobile experience is a heavy compromise.

CLEAN APPLICATION: Apply this to Clean's "Case Studies" or customer-story page. Replace the editorial imagery with terminal-screenshot cards (token streams, TOON output diffs). The ripple shader on hover signals that each artifact is alive — perfect for showing "before TOON / after TOON" payload comparisons. Cursor morph stays.

## 5. Fauna Robotics — https://www.awwwards.com/sites/fauna-robotics

WHAT IT DOES: B2B robotics startup site built in Webflow with restraint — almost zero 3D, all motion comes from precisely timed video loops, SVG icon morphs, and grid-cell reveals. Two-color palette (cream #EFEFEF, ice-blue accent #D1E3FF) with high whitespace ratio. Hero is silent looping video of a robot grip-and-place; type system uses a single sans (likely Söhne or Inter Tight) at four weights. Interaction layer is restrained: 200ms ease-out on hover states, 600ms staggered fade-up on scroll-into-view.

PACKAGES (if rebuilt in code):
- `next@15` or `astro@5`
- `tailwindcss@4`
- `framer-motion@11`
- `embla-carousel-react@8` (gallery)
- `@vidstack/react@1` (video player wrapper)
- Fonts: Inter Tight + JetBrains Mono
- No 3D, no GSAP needed

IMPLEMENTATION:
1. Hero: full-bleed `<video autoplay muted loop playsinline preload="metadata">` with `<source src="hero.av1.mp4" type="video/mp4; codecs=av01">` first, then h264 fallback. Wrap in `aspect-ratio: 16/9` container. Overlay headline at `font-size: clamp(3rem, 6.5vw, 7rem); font-weight: 500; letter-spacing: -0.025em`.
2. Section reveals: framer-motion `<motion.div initial={{ opacity: 0, y: 24 }} whileInView={{ opacity: 1, y: 0 }} viewport={{ once: true, margin: '-10%' }} transition={{ duration: 0.7, ease: [0.16, 1, 0.3, 1], delay: i * 0.08 }} />`.
3. Icon morph: build SVG icons as a single `<path>` per icon; animate `d` attribute with framer-motion `<motion.path animate={{ d: hovered ? pathB : pathA }} transition={{ duration: 0.4, ease: 'easeInOut' }} />`. Pair with stroke-dashoffset draw-in on viewport entry.
4. Grid: CSS `grid-template-columns: repeat(12, 1fr); gap: clamp(16px, 2vw, 32px)`. Each card spans 4 or 6 cols; first card spans 8 for visual asymmetry.
5. Typography hierarchy: only 4 sizes — `text-[clamp(3rem,6.5vw,7rem)]` (display), `text-[clamp(1.75rem,3vw,2.5rem)]` (section), `text-base` (body), `text-xs uppercase tracking-[0.12em]` (eyebrow). Discipline > variety.
6. Color: background `#EFEFEF`, surface `#FFFFFF`, accent `#D1E3FF`, ink `#0A0A0F`. Hover backgrounds shift only luminance (e.g., `#EFEFEF` → `#E8E6E0` at 200ms).
7. Performance: ship under 200KB JS — Astro with islands recommended; lazy-load videos with `loading="lazy"` and `IntersectionObserver` to call `.play()` only on visible.

WHY IT WORKS: Restraint is the message. A robotics company that ships precise hardware can't have a flashy site — it would feel like a contradiction. The two-color palette + whitespace + one-font discipline communicates engineering rigor more credibly than any 3D scene. Hick's law again: every page has one decision.

GOOD / BAD:
- GOOD: This is the highest signal-to-noise ratio of any site this week — it knows what to leave out.
- BAD: The aesthetic is becoming generic (every YC-stage robotics + AI startup ships this exact look) — the differentiation now lives in the content, not the chrome.

CLEAN APPLICATION: Apply this to Clean's docs landing and pricing page. Strip every gradient, every shader, every animation under 200ms easing. Use the same 4-size type system, 2-color palette (Clean ink + cyan accent), `clamp()`-based fluid type, and video loops of TOON token-counts ticking down to communicate compression viscerally. This is the Clean brand register — restrained engineering.

## BUILD DIRECTIVE

TARGET: Clean landing-page hero (tryclean.ai homepage above-the-fold).

APPROACH: Build a Three.js + React Three Fiber hero centered on a single transmissive geometric object — Clean's MCP proxy as a physical artifact light passes through. Tool calls enter from the left as monospace strings, refract through the object, exit on the right as compressed TOON-formatted output. Surround with the restrained Fauna Robotics frame: high whitespace, two-color palette (deep ink + cyan accent), single sans typeface, monospace numerics for context-window stats ("47% fewer tokens," "112ms p50 latency") that tick in via framer-motion `useMotionValue`. Borrow Wolverine's metadata-bar pattern: a fixed bottom-left strip showing `CLEAN PROXY / v0.x / MCP→TOON / EDGE` in JetBrains Mono. Borrow EverSwap's sticky SVG progress path for the section nav as the user scrolls into the rest of the page. Avoid the editorial shader theatrics of Indigo Laboratory — Clean is infrastructure, not fashion.

STACK:
- `next@15`, `react@19`, `typescript@5.6`
- `three@0.168`, `@react-three/fiber@9`, `@react-three/drei@10`, `@react-three/postprocessing@3`
- `gsap@3.13` + `ScrollTrigger`
- `framer-motion@11`
- `lenis@1.1`
- `tailwindcss@4`
- Fonts via `next/font`: Inter Tight (UI) + JetBrains Mono (numerics/metadata) + a single serif accent (Editorial New or Tiempos) only for the H1
- `clsx`, `tailwind-merge`
- No icon library — ship custom SVGs hand-authored to match the proxy aesthetic

REFERENCE URLS:
- https://www.awwwards.com/sites/sui (transmissive 3D object as the brand metaphor)
- https://www.awwwards.com/sites/fauna-robotics (restraint, whitespace, two-color B2B register)
- https://www.awwwards.com/sites/wolverine-worldwide (monospace metadata bar pattern)

KEY CONSTRAINTS:
- Clean is developer infrastructure. The hero must feel like Linear/Vercel/Stripe — not like a Web3 mint page. AVOID: neon gradients, particles for their own sake, marquee scrolls of fake logos, "AI-y" purple-to-pink gradients, glassmorphism cards, Aceternity-library default looks (no Spotlight effect, no Background Beams, no Wavy Background — every one of those reads as "vibe-coded landing page" to senior devs in 2026).
- Bundle budget: hero JS ≤ 220KB gzipped. R3F + drei + three is ~180KB — that is the budget.
- The 3D object must have a static AVIF fallback that visually matches at first paint; do not flash a black canvas.
- Type must read as engineered: max 4 type sizes, max 2 typefaces (sans + mono), use `font-feature-settings: 'ss01','cv11','calt'` on Inter Tight.
- Motion must feel mechanical, not bouncy. Default easing: `cubic-bezier(0.16, 1, 0.3, 1)` (expo.out). Default duration: 600–800ms. No spring physics anywhere on UI chrome.
- Accessibility: respect `prefers-reduced-motion` — kill the 3D animation, freeze the object on the cleanest frame, keep all content reachable.
- Performance: Lighthouse Performance ≥ 92 on mid-tier mobile (Moto G Power). LCP ≤ 1.8s. CLS = 0.
